## 🎯 Pro Tips for Better Recording

### Before Recording

**✅ DO:**
- Close all unnecessary programs
- Close extra browser tabs
- Test your microphone (speak and listen)
- Practice the script once without recording
- Have water nearby for clear voice
- Enable Focus Assist (no notifications)

**❌ DON'T:**
- Rush through the demo
- Forget to unmute microphone
- Have background noise (close windows, silence phone)
- Move mouse too fast
- Skip the practice run

### During Recording

**✅ DO:**
- Speak clearly and at moderate pace
- Pause between sections (2-3 seconds)
- Move mouse slowly and deliberately
- Hover over important items
- Smile while speaking (sounds friendlier!)

**❌ DON'T:**
- Speak too fast
- Move mouse erratically
- Click too quickly
- Forget to pause on important info
- Apologize for mistakes (just continue)

### If You Make a Mistake

**Option 1: Keep Going**
- Pause for 2 seconds
- Take a breath
- Continue from the last good point
- You can edit out mistakes later

**Option 2: Start Over**
- Stop recording (Windows + Alt + R)
- Delete the video
- Start fresh (it's okay!)
- Most people record 2-3 times

## ❓ Troubleshooting

### Problem: Game Bar doesn't open (Windows + G)

**Solution:**
```
1. Press Windows + I (Settings)
2. Go to Gaming → Xbox Game Bar
3. Turn ON "Enable Xbox Game Bar"
4. Try Windows + G again
```

### Problem: No audio in recording

**Solution:**
```
1. Press Windows + G
2. Click Settings (gear icon)
3. Go to Capturing
4. Check "Record audio when I record a game"
5. Select correct microphone
6. Test and try again
```

### Problem: Video quality is poor

**Solution:**
```
1. Press Windows + G
2. Click Settings (gear icon)
3. Go to Capturing
4. Set Video quality to "High"
5. Set Video frame rate to "30 fps"
6. Try recording again
```

### Problem: File is too large (over 100MB)

**Solution:**
```
Option 1: Compress with HandBrake (see above)
Option 2: Record at lower quality
Option 3: Make video shorter (3 min instead of 5)
Option 4: Use cloud storage instead of email
```

### Problem: Made a mistake while recording

**Solution:**
```
Don't worry! Options:
1. Keep recording, edit out mistake later
2. Stop, delete, start over (totally fine!)
3. Pause for 3 seconds, continue from last good point
4. Most people record 2-3 times - it's normal!
```
