## Method 5: Windows Photos App (Built-in!)

### What You Need:
- Windows 10/11 (you have it!)
- Screenshots of demo
- 10 minutes

### Steps:

**1. Take Screenshots**
```
1. Open POWERBI_DEMO_APP.html
2. Press Windows + Shift + S
3. Take 10-15 screenshots of different sections
4. Save to a folder
```

**2. Create Video**
```
1. Open Photos app (Windows + type "Photos")
2. Click: "New video" → "New video project"
3. Name it: "SmartTaskHub Demo"
4. Add your screenshots
5. Set duration for each (5-10 seconds)
6. Add text overlays (optional)
7. Click: "Finish video"
8. Export as MP4
```

**Pros:**
- ✅ Built into Windows
- ✅ No installation
- ✅ Simple interface
- ✅ Free

**Cons:**
- 📸 Need to take screenshots first
- ⏱️ Manual process (10-15 min)
- 🎵 No narration (can add music)
