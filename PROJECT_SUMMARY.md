## 🎯 Technical Specifications

### Frontend
- **Framework:** React 18
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Build Tool:** Vite
- **Animations:** Framer Motion

### Backend
- **Framework:** FastAPI (Python)
- **ORM:** SQLAlchemy
- **Validation:** Pydantic
- **Protocol:** FastMCP
- **Server:** Uvicorn

### Integration
- **API:** Power BI REST API
- **Auth:** Azure AD OAuth 2.0
- **Gateway:** On-premises Data Gateway

### Database
- **Type:** SQLite (development)
- **ORM:** SQLAlchemy
- **Migrations:** Alembic
